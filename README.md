# COFFEE SHOP DOMAIN MODEL

# Project overview
This Python project models is about a coffee shop using object-oriented programming principles. It demonstrate about the relationship between coffees, customers and orders.

The project follows a real-world situations:
- A customer can place multiple orders
- A coffee can be ordered by multiple customers
- An order belongs to a single customer and a single coffee 
- The many-to-many relationship between Customers and Coffees is handeled through Orders

# Installations 
1. Created a repository 
2. Set up the virtual environment within the directory 
- pipenv install
- pipenv shell
3. Installed testing packages
- pipenv install pytest
